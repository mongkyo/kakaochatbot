- OS : macOSX
- 데이터베이스 : postgreSQL
- 배포 : AWS ElasticBeanstalk, kakao i Open Builder
- skills

	- Framework : Django==2.1.5, selenium==3.141.0
	- Library : Beautifulsoup4==4.7.1
	- Cloud : AWS, Docker
	- version control : git, github
